Here are a few basic git commands:

git remote add origin URL // sets the url of the host git repository
git push -u origin master // pushes the commited files to git
git pull origin master    // pull the files and merge to local repository

git config --global user.email EMAIL
git config --global user.name "FULL NAME"

git add filename // stage a file for committing

git commit -m "Descriptive message" // commit changes to local repository

git chekcout -b BRANCH //create a branch (think subsystem)
